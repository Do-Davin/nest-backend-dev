## 📚 About This Repository

This repository is dedicated to **backend laboratory work for the Mobile App Development subject**.

It contains:

- NestJS backend practice
- REST API experiments
- Backend architecture exploration
- Lab exercises for mobile backend services
- Concept demonstrations for server-side development

This repository **does NOT contain a production-ready backend system**.  
It is strictly used for **learning and lab practice**.

---

## ⚙️ Technologies Used

- **NestJS** — Progressive Node.js framework for building scalable server-side applications
- **TypeScript** — Strongly typed JavaScript for better development experience
- **Node.js** — JavaScript runtime for server-side execution

---

## 📂 Repository Purpose

The goal of this repository is to:

- Practice **NestJS backend development**
- Understand **REST API architecture**
- Learn **server-side application structure**
- Experiment with **backend services for mobile apps**
- Document learning progress throughout the course
